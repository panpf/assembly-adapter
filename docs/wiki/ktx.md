# 在 Kotlin 中使用 bindview

从 JCenter 导入 Kotlin 扩展即可：

```groovy
dependencies {
    implementation 'me.panpf:assembly-adapter-ktx:$lastVersionName'
}
```

`$lastVersionName`：[![Release Version][release_icon]][release_link]
